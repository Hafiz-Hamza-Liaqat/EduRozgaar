# Contributing to EduRozgaar

Thank you for your interest in EduRozgaar – Pakistan Career Intelligence Platform.

## Ownership

Project ownership and primary contacts are listed in [AUTHORS.md](./AUTHORS.md).  
Copyright and branding attribution are described in [NOTICE.md](./NOTICE.md).

## License

By contributing, you agree that your contributions are licensed under the same
terms as the project: the [MIT License](./LICENSE).

## Before you start

1. Open an issue to discuss significant changes when possible.
2. Do not commit secrets (`.env`, API keys, credentials, or local database data).
3. Prefer small, focused pull requests with a clear description.
4. Follow existing code style and architecture; do not redesign core Career
   Intelligence scoring or add paid AI dependencies without discussion.
5. Paid AI integrations remain disabled by default — see `docs/AI_BUDGET_POLICY.md`.

## Development basics

See `docs/SETUP_AND_RUN.md` for local setup. High-level verification scripts are
available via `npm run verify:*` at the repository root.
